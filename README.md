## 🚦 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/hamzakyb/hmz-solutions.git
   cd hmz-solutions
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```
